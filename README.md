--------------------------------------------------------------------
O projeto consiste em um aplicativo de Quiz educacional multidisciplinar, desenvolvido em C# (Windows Forms), com perguntas sobre Matemática, Linguagens, Ciências Humanas e Ciências da Natureza. O sistema permite que o usuário crie uma conta, realize login, jogue quizzes das diferentes áreas e acumule pontuações a cada resposta correta.

Além disso, o jogo conta com um sistema de Daily Streak, que registra a sequência de dias consecutivos que o usuário acessa o aplicativo, recompensando sua constância. As pontuações obtidas são armazenadas e exibidas posteriormente em um placar, permitindo que o jogador acompanhe seu desempenho.

O objetivo do projeto é promover aprendizagem interativa, incentivando a prática diária e o estudo divertido por meio de desafios de perguntas e respostas.
--------------------------------------------------------------------
 1. Login e Cadastro
Ao iniciar o aplicativo, o usuário encontra a tela de Login.
Caso não possua conta, deve selecionar Cadastrar, preencher os dados solicitados e confirmar.
Após o cadastro, é possível acessar o sistema normalmente utilizando login e senha registrados.

 2. Como Jogar
Após o login, o usuário será direcionado ao Menu Principal.
Basta selecionar uma das áreas do conhecimento:
Matemática
Linguagens
Ciências Humanas
Ciências da Natureza
Cada área possui um conjunto de perguntas de múltipla escolha.
Clique na alternativa desejada para responder.
Ao final, o sistema indica Vitória ou Derrota, de acordo com o desempenho.

 3. Pontuação e Ranking
Cada resposta correta gera pontuação.
A pontuação acumulada é registrada automaticamente no sistema.
O usuário pode consultar o placar, onde estarão registradas suas pontuações.

 4. Daily Streak
O sistema contabiliza o número de dias consecutivos em que o jogador acessa o aplicativo.
O streak é mostrado em uma tela específica e funciona como incentivo de constância.
Para manter o streak, basta logar pelo menos uma vez por dia.

5. Erros Comuns (Soluções)
Problemas e	Possívis Soluções:
Não consigo logar:	Verifique usuário e senha, e se já realizou cadastro.
Pontuação não aparece no ranking:	Reinicie o aplicativo após finalizar uma rodada.
Streak não aumenta:	Certifique-se de que jogou/logou em dias consecutivos.



Obrigado por jogar, boa sorte!
